# Heartbeat of Love - A Web Development Project

## Overview

**Heartbeat of Love** is a single-page web project created for the AWSCC-PUP DWD Week 4 Challenge. This project showcases a beautifully animated heart that symbolizes love, accompanied by interactive features such as love quotes, floating hearts, and theme customization. Built using **HTML, CSS, and JavaScript**, this project demonstrates the creative use of front-end web development tools to create a visually appealing and interactive digital art piece.

---

## Features

1. **Beating Heart Animation**  
   - A heart shape created using CSS transforms and pseudo-elements.
   - Smooth, continuous heartbeat animation using CSS `@keyframes`.

2. **Interactive Love Quotes**  
   - A collection of love quotes that cycle with each click on the heart.
   - Smooth fade-in and fade-out transitions for quote changes.
   - Users can copy quotes to the clipboard with a single click.

3. **Floating Hearts Background**  
   - Randomly generated floating hearts in the background for a romantic ambiance.
   - Toggleable feature to enable or disable the floating hearts.

4. **Customizable Themes**  
   - Light and dark themes to suit user preferences.
   - Toggle between themes with a single button click.

5. **User-Friendly Controls**  
   - Buttons to navigate between quotes (previous and next).
   - Option to add custom love quotes to the collection.

6. **Responsive Design**  
   - Optimized for various screen sizes, ensuring a seamless experience on both desktop and mobile devices.

---

## Technologies Used

- **HTML**: For structuring the webpage.
- **CSS**: For styling, animations, and transitions.
- **JavaScript**: For interactivity and dynamic content updates.

---

## How to Use

1. **Interact with the Heart**  
   - Click the heart to cycle through love quotes.
   - Hover over the heart to see it scale up slightly.

2. **Navigate Quotes**  
   - Use the "Previous" and "Next" buttons to manually navigate through the quotes.

3. **Add Your Own Quote**  
   - Click the "Add Quote" button to input your own love quote, which will be added to the collection.

4. **Copy Quotes**  
   - Click on any displayed quote to copy it to your clipboard.

5. **Toggle Themes**  
   - Use the "Toggle Theme" button to switch between light and dark themes.

6. **Floating Hearts**  
   - Use the "Toggle Hearts" button to enable or disable the floating hearts in the background.

## Installation

1. Clone the repository or download the project files.
2. Open the `index.html` file in your preferred web browser.
3. Enjoy the interactive experience!

## Customization

- **Add More Quotes**: Edit the `quotes` array in `script.js` to include your favorite love quotes.
- **Change Colors**: Modify the CSS variables in `style.css` to customize the color scheme.
- **Adjust Animations**: Tweak the `@keyframes` and transition properties in `style.css` to create different animation effects.
